# 🏥 Hospital Analytics Dashboard | Power BI

## 🚀 Transforming Hospital Data into Powerful Insights

Healthcare organizations generate huge amounts of data from multiple sources such as patient records, financial systems, medical inventory, and operational databases.

Managing this scattered data makes it difficult to understand hospital performance and take data-driven decisions.

To solve this problem, I developed a **Hospital Analytics Dashboard using Power BI** that integrates healthcare data into a single interactive reporting platform.

---

# 📊 Project Overview

The Hospital Analytics Dashboard provides a complete 360° view of hospital operations including:

- Patient Management
- Doctor Performance
- Surgery Tracking
- Medicine Inventory
- Financial Analysis
- Hospital Resource Utilization


---

# 🎯 Key Objectives

✔ Analyze patient admission and discharge patterns

✔ Monitor bed availability and occupancy rate

✔ Track doctor performance and commission

✔ Analyze surgery and test status

✔ Monitor medicine stock and sales

✔ Understand hospital revenue and billing performance
